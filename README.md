# AI-Powered Virtual Teaching Assistant

## Problem Statement
Students face difficulty in getting instant answers to academic doubts.

## Solution
This project is a chatbot that answers student questions using NLP and ML.

## Features
- Question answering system
- Keyword extraction using TF-IDF
- Fast response using cosine similarity

## Tech Stack
- Python
- NLTK
- Scikit-learn

## Installation
pip install -r requirements.txt

## Usage
python main.py
